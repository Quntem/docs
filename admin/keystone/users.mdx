this page will cover how to manage users in Quntem KeyStone.

## View All Users

To view all users, navigate to the "Users" tab in the Admin Center or Team Dashboard. You should see a list of all users in your tenant. 

## Edit a User

From the list of users, click on the user you want to edit. You should see a form with the user's information. You can edit the user's information here. Once you are finished, click "Save".

### Change Password

In the "Quick Actions" section, click "Password". You should see a form to reset the user's password. You can change the user's password here. Once you are finished, click "Confirm".

### Disable User

In the "Quick Actions" section, click "Disable". You will be asked to confirm this action. This will disable the user's account. The user will not be able to log in to Quntem products until their account is enabled again.

<Note>Make sure you do not disable the last admin user in your tenant. This will prevent you from accessing the Admin Center or Team Dashboard. If you do, you will need to [contact Quntem Support](mailto:support@quntem.co.uk). Make sure to include your domain name and proof of ownership for the domain.</Note>

## Create a User

To create a user, navigate to the "Users" tab in the Admin Center or Team Dashboard. Click the "Add User" button. You should see a form to create a new user. Fill in the form and click "Add".
